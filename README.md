# WeVo Pay

WeVo Pay is a P2P platform that helps users safely exchange money between mobile wallets (Vodafone Cash, Orange Money, etc.) and InstaPay. 

Instead of dealing with unknown middlemen on social media, users can create offers, get matched with someone who has the opposite need, and complete the transaction inside a secure Trade Room after both sides upload proof of payment.

## Features

- User registration with OTP verification
- KYC process (National ID upload + Live Capture)
- Create and browse offers in the market feed
- Private Trade Room with receipt upload
- Transaction confirmation and dispute system
- Earnings tracking and referral system

## Tech Stack

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views + Bootstrap



## Project Status

This is the first working version focusing on the core matching and escrow flow between wallets and InstaPay.
